# Results (2026-02-12)

## 1) Stability benchmark (event_time)

Command:

```bash
python experiments/exp13_stability_protocol/stability_benchmark.py --strat-mode event_time --cv-seeds 1,7,21,42,2026 --n-splits 5 --n-repeats 10 --quiet-cv
```

Key outputs:

- RSF-only baseline: `mean=0.97312`, `std=0.00032`
- Fixed blend (RSF=0.9): `delta_mean=-0.00021`, `delta/noise=-0.67`
- OOF-opt blend: always `w_RSF=1.00`

Decision:

- In OOF, forcing `RSF=0.9` is a stable negative contribution.
- Keep RSF cap configurable; do not force cap by default.

## 2) Isomorphic OOF (event vs event_time)

Command:

```bash
python experiments/exp13_stability_protocol/isomorphic_oof_eval.py --compare-strat --cv-seed 42 --n-splits 5 --n-repeats 10 --quiet-cv
```

Summary:

- `event` strat: RSF(pp) `0.9723`
- `event_time` strat: RSF(pp) `0.9735`
- Gain from `event_time`: `+0.0012`

Decision:

- Use `event_time` as default stratification mode in training.
