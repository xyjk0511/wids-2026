  <name>Task 1: Implement IPCW stacking experiment script</name>
  <files>scripts/exp31_ipcw_stacking.py</files>
  <action>
Create scripts/exp31_ipcw_stacking.py as a standalone experiment script.

IPCW weight computation:
- Use lifelines.KaplanMeierFitter to estimate censoring distribution G(t)
- Fit KM on (y_time, 1-y_event) to model censoring
- Weight for event=1: 1/G(t_i), clipped min=0.05. Weight for event=0: 1.0
- Print weight distribution stats (min/max/p5/p95)

Base model pool (3 models, 12 meta-features = 3 x 4 horizons):
- RSF(n_estimators=200, min_samples_leaf=5)
- EST(n_estimators=200, min_samples_leaf=5)
- GBSA(n_estimators=300, max_depth=3, learning_rate=0.02) -- instantiate directly from src.models, NOT via _make_base()

Meta-learner comparison:
- Ridge(alpha=1.0) with sample_weight=IPCW, clip output to [0,1]
- LogisticRegression(C=0.01, max_iter=2000) with sample_weight=IPCW
- Per horizon h in [12,24,48,72]: build labels via build_horizon_labels(), apply eligible mask, fit on eligible samples only
- Include 72h meta-features (Ridge handles near-constant columns automatically)

Two-stage CV (per CONTEXT.md locked decision):
- Stage 1: RepeatedStratifiedKFold(n_splits=5, n_repeats=1)
- Each outer fold: train 3 base models, collect OOF as 12 meta-features
- Fit both Ridge and LR per horizon with IPCW weights
- If best OOF hybrid <= 0.9697: print "No signal -- stopping" and exit
- Stage 2 (only if Stage 1 passes): n_repeats=10, same procedure

Go/no-go gate (per CONTEXT.md locked decision):
- Load submissions/submission_0.96624.csv as reference
- Spearman correlation between OOF and reference per horizon
- Gate passes if: OOF hybrid > 0.9697 AND all Spearman > 0.90

Environment: .venv_sksurv22. Do NOT modify src/train.py. Do NOT add GBSA to _make_base().
