  <files>scripts/exp30_blend_anchors.py</files>
  <action>
Create blending script with blend admission gate + OOF-driven weight search:

**Blend admission gate** (check before any blending):
- Gate 1: new anchor LB > 0.96624 (main gate — must pass)
- Gate 2: rho48 ∈ [0.90, 0.99] (too correlated = no gain; too different = risky)
- Gate 3: distribution not collapsed (p48 std > 0.01, no >80% values at boundary)
- If any gate fails: print which gate failed and exit without producing blend CSVs

**Value-space weighted blend** (if all gates pass):
- Accept CLI args: --sub_a (path) --sub_b (path) --oof_a (path) --oof_b (path)
- Coarse weight search: w ∈ [0.3, 0.4, 0.5, 0.6, 0.7] (5 points)
- For each w: blended[col] = w * sub_a[col] + (1-w) * sub_b[col] (value space, not rank)
- Score each blend via OOF: compute hybrid OOF score using oof_a, oof_b at weight w
- Narrow search: take best coarse w ± 0.05 in steps of 0.01 (11 points)
- Apply submission_postprocess (monotonicity + clip) to final blended output
- Save top-3 blends as submissions/exp30_blend_w{w}.csv
- Print OOF score + Spearman(blend_p48, ref_p48) for each candidate

Per CONTEXT.md: only allow submissions that change p48 ranking with stability evidence.
  </action>

If gates pass, script produces blend CSVs with OOF scores printed.
  </verify>
